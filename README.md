# Imagine Vision

The Imagine marketing website — configurable commerce platform.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
npm start
```

## Project structure

- `app/` — Next.js App Router pages
- `components/` — Shared UI components
- `docs/` — Strategy and page specs
- `legacy/` — Archived static HTML (design-system, etc.)
