Design System Thinking

Foundations for the Imagine 2026 experience — color, type, spacing, and components derived directly from the Figma source file.

Overview

Imagine 2026 uses a dark, minimal visual language built on a single brand accent — Papaya orange — against a near-black surface. The palette is intentionally constrained to keep focus on content.

All tokens below are sourced from the Figma file. Use CSS variables throughout; never hard-code raw values.

Typography

PP Neue Montreal is the sole typeface. Use weight 400 for body, 530 (medium) for display, and 600 for headings and labels.

Guidelines

  • Dark surfaces first. Default to the dark surface palette. Only use light surfaces for specific print or embed contexts.
  • Papaya is the only accent. Do not introduce secondary brand colours. Papaya signals every interactive or focal element.
  • PP Neue Montreal only. Do not mix in other typefaces. Use weight, size, and letter-spacing for hierarchy.
  • 8px grid. All spacing must be a multiple of 8px (or 4px for sub-pixel alignment). Avoid arbitrary values.
  • Contrast. Body text (#FCFCFC on #141414) exceeds WCAG AAA. Maintain at least AA for all interactive text.
  • Touch targets. Minimum 44×44px for all interactive elements on touch devices.
